本文主要介绍在性能优化中用到的几个工具：

# Android  Monitor

可以实时监测内存、CPU 、GPU 、网络状态的变化
# Mat

强大的内存分析工具

# LeakCanary

内存泄露检测的第三方库

# TraceView

查看内存中的某个线程的内存的分配情况，检测内存抖动

# Hierarhy view

检测布局的过度绘制

# Allcation Tracing

跟踪内存分配情况

# Lint

排除应用中的重复、无用资源代码；检测内存泄露；有问题的代码

# Android 终端设备Debug GPU overdraw

检测过度绘制，直接通过视觉反应
